# GetUp
Script notifies that you need to get up to your set.

# Execution
<ol>
  <li>
    <p>
      Allow the script to be executed
    </p>
      <b>chmod +x ./getup</b>
  </li>
  <li>
    <p>
      Execute every 60 minutes or one hour 
    </p>
      <b>./getup 60 </b>
  </li>
</ol>

# Build
<p>
  <b>go build getup.go</b>
</p>
